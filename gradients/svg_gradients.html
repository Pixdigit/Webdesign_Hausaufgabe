Einleitung

SVG Gradienten sind Farbverläufe innerhalb von SVG, dabei ist die grundlegende Idee einen solchen Farbverlauf vorher zu definieren und dann auf jeden beliebigen SVG fill an zu wenden. So besitzt der Gradient Koordinaten, welche Angeben in welche Richtung der Farbverlauf gehen soll, sowie den Stil des Farbverlaufes und die Eigenschaften der Farbe wie Farbton oder Durchsichtigkeit. Dabei muss darauf geachtet werden dass die Beschreibung des Gradienten in einer besonderen <defs> Anweisung geschieht, welche laut W3 für die Definition von besonderen Elementen vorgesehen ist.

Einen Gradienten definieren

Die Defintion eines Gradienten beginnt mit der <linearGradient> oder der <radialGradient> Anweisung. Darauf folgend bekommt jeder Gradient eine einzigartige ID zugewiesen indem id="" im Gradient Block definiert wird. Darauf folgen die Koordinaten des Farbverlaufes welche abhängig von der Art des Gradienten entweder in x1, y1, x2, y2 für lineare Gradienten oder in cx, cy, r, fx, fy für radiale Gradienten definiert werden. Daraufhin wird in <stop>-tags definiert an welchen Stellen (in %) der Gradient welche Eigenschaften, wie Farbe oder Durchsichtigkeit annehmen soll.

Einen Gradienten auf SVG anwenden

Nachdem ein Gradient in den <defs>-tags definiert wurde kann er auf den "fill" - teil einer SVG Definition angwendet werden indem statt einer Farbe angegeben wird: fill="url()" wobei der Inhalt der Klammern die eindeutige ID des Gradienten ist.
