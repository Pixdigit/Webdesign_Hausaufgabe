Einleitung

SVG benutzt geometrische Figuren, sog. "Shapes", um Grafiken zu erzeugen. Shapes können beliebig kombiniert und überlagert werden. Dargestellt wird aber immer in Leserichtung. Also eine in der Textdatei weiter unten plazierte Shape überlagert alle, die davor definiert wurden. Im folgenden nun einige Beispiele zu den wichtigsten.

Für eine Vollständige Referenz, schaue hier.

Rechteck

Eines der einfachsten grafischen Primitive. Es besteht nur aus einer Position sowie einer Breite und Höhe. In der Animation werden die vier Parameter verändert, wodurch ihre Auswirkung deutlich wird.
In SVG wird ein Rechteck so repräsentiert:
<rect x="X-KOORDINATE" y="Y-KOORDINATE" width="BREITE" height="HÖHE">
(Die Großbuchstaben werden natürlich durch entsprechende Werte ersetzt)

Kreis

Wie das Rechteck ist auch dieses grafische Primitiv sehr einfach. Es hat nur zwei Koordinaten für den Mittelpunk und einen Radius. In der Animation werden die drei Parameter dargestellt.
In SVG wird ein Kreis so repräsentiert:
<circle cx="MITTELPUNKT-X-KOORDINATE" cy="MITTELPUNKT-Y-KOORDINATE" r="RADIUS">
(Auch hier werden die Großbuchstaben natürlich durch entsprechende Werte ersetzt)

Ellipse

Die Ellipse ist fast identisch zum Kreis. Sie hat hingegen aber zwei Radien. Jeweils einen für x- und y-Ordinate. In der Animation wird gezeigt, was die zwei Radien bewirken.
In SVG wird eine Ellipse so repräsentiert:
<ellipse cx="MITTELPUNKT-X-KOORDINATE" cy="MITTELPUNKT-Y-KOORDINATE" rx="X-RADIUS" ry="Y-RADIUS">
(Auch hier werden die Großbuchstaben natürlich durch entsprechende Werte ersetzt)
